Le dossier "Pages-components" regroupe tout les composants principaux des chaques pages.

Par exemple : Dans le dossier MesCodes_components se trouves tout les composant principaux pour la page "MesCodes".

A l'intérieur du dossier "MesCodes_components" on peut trouver "CodeCards" qui est le composants qui affiche les card crées par l'utilisateur.

Tout ces sous components (comme "CodeCards") sont utilisé dans un components parent comme "ContainerCode" qui est le component princiapl de "MesCodes_components".

Cela me permet d'obtenir une meilleur organisation et une meilleur vision de code.

Pour chaque dossiers (ex : MesNotes_components) ce sera toujours la même façon de procèder.
